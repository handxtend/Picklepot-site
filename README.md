# PicklePot Frontend

Static frontend for **PicklePot** (Create/Join pots, organizer tools).  
Deployed on **Netlify**; talks to the backend at **Render**.

## Live
- **App:** https://picklepotters.netlify.app
- **API_BASE:** https://picklepot-stripe.onrender.com

## Quick Start (local)
No build step—just open `index.html` or serve the folder:
```bash
# Optionally run a tiny server for CORS-friendly testing
python -m http.server 8080
# visit http://localhost:8080
